🏰 Kingdom of Agents

A live, medieval-themed dashboard for the GitHub Copilot CLI.

Watch your active Copilot CLI sessions become a living kingdom of districts — forge, library, terminal keep, signal tower, guild hall, tome hall, and royal court — with tool calls, agent delegations, errors, and alerts surfaced in real time.
